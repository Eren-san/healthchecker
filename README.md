# Java-based Health Checker Application

A simple Java application that checks the availability and response time of given endpoints.

---

## Usage

### 1. Docker

Build and run the application in an isolated Docker environment.

```bash
# Build the Docker image
docker build -t healthchecker .

# Run the container
docker run healthchecker
```

---

### 2. Manual Execution

Run the application directly using the Java compiler.

```bash
# Compile
javac src/App.java

# Run
java App.java
```
